## About

A browser-based arcade game built as a personal project. 

This repo contains only the compiled build (npm run build)

The source code is not included, as the game is intended for future release in the public domain.

---

## Modes

- **Mouse mode** — point and click to hunt targets
- **Keyboard mode** — type words to destroy targets before time runs out

---

## Controls

- **Pause** — space bar
- **f1** — debug mode

---
